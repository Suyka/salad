# 🥗 Salad

AI時代に最適化されたヘルシー(軽量)なサブブラウザ。

## 機能の概要

AIが検索結果を個人の関心に沿って2〜3記事に厳選・再構成するサブブラウザ。
絞り込み機能(Dressing)と情報熟成機能(Crock)を中核に、
情報過多を防ぐデジタルウェルネス設計を採用。

## 主な機能

- **Dressing** : 検索結果をAI(Shake)が自分好みに厳選・再構成
- **Crock** : 気になった記事を保存し、AIが時間をかけて関連付ける熟成機能
- **Wellbeing** : 長時間使用時にAIが休憩を促す
